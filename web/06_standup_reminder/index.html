06 久坐检测：蹭蹭

这一页专门服务“久坐检测：蹭蹭”场景。重点不是提醒本身，而是让提醒方式像宠物一样可爱：先转向你、三次蹭蹭、两次点头、被拒绝后轻轻摇头。

动作节奏

当前代码按 PDF2 细化成“埋头 -> 顶起 -> 后退”的三次蹭蹭，再补双点头和“好吧”式轻摇头。

  1. 先把底座和灯臂朝评委方向送过去。
  2. 第一次蹭蹭：先往下埋，再向上顶，然后退一点。
  3. 第二次重复同样节奏，维持可爱的“蹭”感。
  4. 第三次蹭完后，不要立刻停，给观众一个节奏上的完成感。
  5. 评委问“你是要我起来吗？”之后，清晰地点两次头。
  6. 评委说“不要”之后，轻轻摇头一下，再慢慢回到原位。

导演提示

这个场景最怕做成“乱抖”。要让观众看出来每一次都像宠物真的在蹭你。

触发条件主持人引导“假装你已经坐了一小时”
核心情绪提醒，但可爱不冒犯
最关键动作三次蹭蹭 + 双点头 + 轻摇头
依赖基础姿态已校准
