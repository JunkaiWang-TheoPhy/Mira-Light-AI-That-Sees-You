09 挥手送别

这一页对应“评委走了，挥手送别”场景。当前代码版本已经有固定角度版：先目送、再慢慢点头两次像挥手、最后低头表示舍不得。

动作节奏

这个场景现在最重要的是把“目送 -> 挥手 -> 低头”分成清楚的三段。

  1. 先把灯头朝评委离开的方向看过去，像在目送。
  2. 再做两次慢慢点头，替代“挥手”的动作感。
  3. 最后微微低头，像有点舍不得。
  4. 收尾回到中性位，但保持更柔和的低亮度。
