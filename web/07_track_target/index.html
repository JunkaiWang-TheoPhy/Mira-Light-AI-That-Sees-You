07 追踪（展示感知能力）

这一页对应“追踪”场景。当前代码实现的是展位排练用 surrogate choreography：让观众先看懂“它在跟”，但它还不是最终的真实视觉闭环。

当前已实现的节奏

先做出左 -> 中 -> 右 -> 停 -> 再跟的追踪节奏，用来排练展位表演语言。

  1. 书在左侧时，灯头压低看向左边桌面。
  2. 目标往中间移动时，灯头跟回中间。
  3. 目标继续到右侧时，灯头继续跟到右边。
  4. 评委停下来时，Mira 也停住。
  5. 评委再次移动时，Mira 再次跟上。
  6. 收尾回到中性工作位。
